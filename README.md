# Wedding Company Quiz Assignment

Interactive quiz application built with Next.js, React, Framer Motion, and Tailwind CSS.

## Features

- 4-question interactive quiz
- Smooth animations with Framer Motion
- Gradient backgrounds and glassmorphism effects
- Custom typography (DM Serif Display & Manrope)
- Animated score counter
- Fully responsive design

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- React Icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
app/
├── layout.tsx      # Root layout with fonts
├── page.tsx        # Main quiz component
└── globals.css     # Global styles
```
